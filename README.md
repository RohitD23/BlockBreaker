# BlockBreaker
Block Breaker is a game inspired by **Breakout** released on Atari console in 1976.
This project is made with the help of **OpenGL**.

### Dependencies Used
1. **GLFW**
2. **GLEW**
3. **GLM**
4. **irrKlang**
5. **FreeType**
